# Saxophone Edu

Static saxophone fingering charts and SVG/HTML fingering animation for alto, tenor, and soprano saxophone.

## Live Site

https://saxophone-edu.vercel.app

## Contents

- `index.html` - landing page
- `sax-fingering-animation.html` - interactive fingering animation
- `sax-fingering-animation.svg` - standalone SVG animation
- `*-sax-fingering-chart.html` - printable fingering chart pages
- `*-sax-fingering-chart.svg` - vector poster charts
- `*-sax-fingering-chart.pdf` - printable PDF charts
- `generate-alto-sax-fingering-chart.mjs` - chart generator
